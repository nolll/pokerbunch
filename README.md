# pokerbunch

Frontend for pokerbunch.com

## Local development

- Clone backend project `https://github.com/nolll/pokerbunch-api.git`
- Make sure backend and database is running. See readme of backend project
- Clone frontend project `https://github.com/nolll/pokerbunch.git`
- `npm install`
- `npm run serve`
- Head to `http://localhost:9010`
- Login with a dev user, See readme of backend project
